# 与你遨游天际 H5

这是一个可直接分享链接的手机网页版本。

## 本地预览

直接双击打开 `index.html` 就能看效果。

## 用 Vercel 发成链接

1. 打开 [Vercel](https://vercel.com/)
2. 用 GitHub 登录
3. 点 `Add New...`
4. 选 `Project`
5. 选中你放这个网页代码的仓库
6. 保持默认配置直接点 `Deploy`
7. 部署完成后会得到一个 `https://xxx.vercel.app` 链接
8. 把这个链接发给对方，微信里就能直接打开

## 说明

- 音乐是手动点击播放，符合微信里常见的交互限制
- 使用的是可在线播放的浪漫钢琴曲，不是《告白气球》原曲
